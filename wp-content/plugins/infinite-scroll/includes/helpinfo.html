This screen allows you to change the settings for the Infinite-Scroll plugin.

The most common issues that users face when trying to set this plugin up are either:

  • They don't know what selectors to use.
  • The plugin doesn't work correctly with a certain theme or plugin. (In rare cases it might even break them!)

If you are having either of the problems above or perhaps something different then fear not! We actively monitor the forums over at Wordpress Support. Remember, your issue will be looked at/rectified faster if you provide all or some of the following info:

  • Version of the script you are using. HINT: You are currently running version: {INFSCROLL_VERSION}
  • The exact issue you are seeing, with as much detail as possible. Its near impossible to help when you open a thread screaming "IT DOESNT WORK!! PLZ HELP. KTHNX.<end post>".
  • An example that we can look at, be it the url to the site you are working on or at the very least the combination of theme/plugins that you are running.
  • Use common sense, if theres additional info that you think will help us help you then put it in the post.
